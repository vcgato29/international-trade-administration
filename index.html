My name is Kin Lane, this is my online domain where I work to understand the world of the Application Programming Interfaces, also known as APIs. This new way of sharing data using the web is touching almost every aspect of our increasingly digital lives, providing access to the bits and bytes that make our personal and professional worlds go round.

This is my work to profile the {{ site.title }}, and generate machine readable definitions for the APis using APIs.json, OpenAPI, and Postman Collections. Making the APIs provided by {{ site.title }}, much more discoverable, and usable beyond just their API portal.

API Evangelist is a network of data driven projects and APIs which I curate and manage as part of this ongoing research, hoping to provide easy access to the moving parts of my work. Everything you see here runs on Github, making everything forkable, and resuable for both humans and machines.

About This Project

This project is part of a larger efort to make APIs more discoverable and usable by making sure there are machine readable definitions available for the entire surface area of the API, and it's supporting operations.

If you'd like to get involved you can visit the Github repository behind this project, and participate by submitting pull requests, and issues for the project, contributing to the definitions it contains.
